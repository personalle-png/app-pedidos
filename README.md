# Sistema de Pedidos

## Como usar

1. Instale as dependências:
   npm install
2. Crie um arquivo `.env` com base no `.env.example`
3. Rode localmente:
   npm run dev
4. Para publicar no Vercel, envie a pasta inteira ao GitHub e configure:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY

## Build

npm run build
